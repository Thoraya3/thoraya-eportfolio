## Task B – Logic Circuit Design: Simple Alarm System

Explanation of Circuit Functionality:

The alarm system is implemented using the Boolean OR operation. In Boolean logic, the OR operator evaluates to true when at least one of its input values is true, which makes it well suited to alarm and monitoring scenarios where multiple independent conditions can trigger an alert. The circuit continuously evaluates the states of the motion sensor and the door sensor. If motion is detected or the door is open, the output signal activates the alarm.

The use of a truth table ensures that all possible combinations of input values are considered and confirms that the circuit behaves correctly under every operating condition. Simple logic circuits such as this are widely used in real-world computing and engineering applications, including security systems, access control mechanisms, and automated monitoring solutions. This logic is similar to alert conditions used in monitoring and automation systems within telecom environments.
